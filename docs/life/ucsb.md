# UCSB 交换

这里会记录 UCSB 交换相关内容：出发前准备、课程体验、校园生活、旅行路线和一些真实感受。

## 准备清单

- 选课和课程难度。
- 住宿、交通和日常开销。
- 手机卡、银行卡、保险等生活事项。
- 行李准备和到达后的第一周安排。

## 交换记录

后续可以按时间线整理：

| 阶段 | 记录内容 |
| --- | --- |
| 出发前 | 申请、签证、选课、行李 |
| 刚到达 | 入住、熟悉校园、办卡 |
| 学期中 | 课程、作业、社交、旅行 |
| 离开前 | 总结、遗憾、建议 |
