# 夏天的个人网站

欢迎来到这里。

这是一个用来整理学习、项目和生活记录的个人网站。它会从一个简单的笔记仓库慢慢长出来：统计学习、编程实践、足球观察、UCSB 交换经历，以及一些读书观影后的想法，都会陆续放在这里。

## 现在可以看什么

| 栏目 | 内容 |
| --- | --- |
| [关于](about.md) | 网站定位、个人简介和写作原则 |
| [学习笔记](learning/index.md) | 统计、概率、数学工具和课程复盘 |
| [编程](coding/index.md) | Python、数据分析、建站记录和小项目 |
| [生活](life/index.md) | 足球、UCSB 交换、书影音和日常观察 |
| [更新日志](changelog.md) | 网站结构和内容的变化 |

## 最近会补的内容

- 统计学习笔记：把作业和课程里容易混淆的概念整理出来。
- Python 数据分析：记录常用代码片段、图表和复现流程。
- UCSB 交换记录：整理准备清单、课程体验和生活建议。
- 足球随笔：写一些比赛复盘和赛季观察。

## 写在前面

我希望这个网站保持一种朴素但持续的节奏：内容可以慢慢补，结构尽量清楚；文章可以不长，但最好真诚、具体，并且能在以后帮我重新理解当时的想法。
