# 学习笔记

这一部分用来放课程笔记、统计学习整理、数学工具和一些概念复盘。我的目标不是把教材搬过来，而是把「我真正理解了什么」写清楚。

## 当前方向

- 统计学：抽样、估计、检验、回归、贝叶斯思路。
- 概率论：随机变量、分布、期望、极限定理。
- 数据分析：从问题定义到清洗、建模、解释结果。
- 数学工具：线性代数、微积分、优化方法中常用的部分。

## 推荐阅读顺序

1. 先看「统计学习」，了解这里会怎样整理统计相关内容。
2. 再看「数学工具箱」，补上公式、推导和常用方法。
3. 后续具体文章会按主题继续拆分。

!!! note "维护计划"
    之后每完成一章课程或一个作业主题，我会把其中最容易混淆的点整理成短文，尽量配上可运行代码和中文解释。
