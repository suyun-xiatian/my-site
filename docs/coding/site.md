# 建站记录

这个网站使用 MkDocs 搭建，主题采用 Material for MkDocs。它适合写个人笔记、课程整理和项目文档，因为 Markdown 文件本身就很轻便。

## 当前结构

```text
docs/
  index.md
  about.md
  learning/
  coding/
  life/
mkdocs.yml
```

## 本地预览

```bash
.venv/bin/mkdocs serve
```

运行后可以在浏览器中打开本地地址预览。
