# 项目与实践

这里收集能够运行、复查和继续迭代的实践项目。每个项目会尽量说明问题、数据、方法、结果与下一步，而不只展示最终截图。

## 当前项目

### suyunax-site

个人知识库与长期写作网站，使用 Zensical 构建并通过 GitHub Pages 发布。

- **目标**：把学习笔记、代码实践和生活记录组织成可持续维护的网站。
- **技术**：Markdown、Zensical、CSS、GitHub Actions。
- **当前状态**：基础信息架构和视觉系统已经建立，正在补充内容发现、公式和图片能力。
- **源码**：[suyun-xiatian/my-site](https://github.com/suyun-xiatian/my-site)

## 项目记录模板

后续项目会统一记录以下信息：

1. 要解决的问题和使用场景。
2. 数据或输入从哪里来。
3. 为什么选择当前方法。
4. 如何复现运行结果。
5. 结果是否可靠，有哪些限制。
6. 下一次迭代准备改进什么。
